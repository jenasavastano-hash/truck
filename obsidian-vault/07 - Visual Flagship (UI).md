# Визуальный флагман (UI)

Цель: **не «тестовый скин»**, а уровень **флагманского продукта** — цельность бренда, читаемость, тактильные детали, спокойная иерархия. Демо с мультяшными фонами — задел; **прод** = сдержаннее, но так же запоминающе.

## Примитивы в коде

В `frontend/src/index.css` (слой `@layer components`): классы **`freight-card`**, **`freight-btn-primary`**, **`freight-btn-primary-compact`** (тулбары, чипы), **`freight-btn-secondary`**, **`freight-input`** — опорная точка для выноса дублей; новые экраны и правки по возможности используют их вместо копирования длинных tailwind-цепочек.

## Принципы

1. **Один визуальный язык** — палитра, скругления, тени, типографика согласованы на логине, home, всех кабинетах, модалках.
2. **Грузовой контекст без клоунады в данных** — иллюстрации и фон не перекрывают формы и таблицы; контраст текста WCAG-разумный.
3. **Роль видна за 1 секунду** — шапка, иконография, заголовки: водитель / менеджер / админ / директор не «один серый экран».
4. **Состояния** — loading / empty / error / success оформлены, не голый текст.
5. **Плотность** — информационные экраны (таблицы ЭПЛ) отдельно от «маркетинговых» (лендинг-ощущение на home).

## Опорная палитра (текущая линия)

- Фон: slate / sky / **teal accent** (не такси-жёлтое).
- Акцент действия: teal/emerald; опасность — красный осознанно.
- Фоны кабинетов: единый компонент сцены + не перегружать анимациями при работе с формами.

## Чеклист «флагман готов»

### База

- [ ] Все экраны с `freight-panel-bg` / backdrop — без артефактов скролла и z-index.
- [ ] Шапки и табы не перекрывают дропдауны (порталы, z-index согласованы).
- [ ] Мобильная ширина: нет горизонтального скролла на ключевых страницах.

### Полировка

- [ ] Единый компонент кнопки primary/secondary/ghost.
- [ ] Модалки: фокус-ловушка, закрытие по Esc/оверлей, анимация 150–200 ms.
- [ ] Пустые списки: иллюстрация или иконка + CTA, не «пусто».

### Бренд

- [ ] Favicon + title + meta description — «Грузовые ЭПЛ», без такси в прод-строках.
- [ ] Скриншоты для презентации: home + кабинет водителя + админ (3 кадра минимум).

## Не смешивать

- «Детский» тестовый фон на экранах с тяжёлыми формами — либо приглушить, либо отдельный пресет «презентация».
