# Визуальная сцена — логин и небо (эпик)

**Статус:** реализация в коде (`frontend/src/components/freight/FreightNightSkyLayers.jsx`, `frontend/src/pages/Login.jsx`).  
**Связь:** [[06 - Freight Rebrand & Demo]], дорожная карта продукт — [[04 - Roadmap]].

## Цель

Единая «киношная» промзона: ночь/день, читаемый американский semi без брендов, небо без перегрузки.

## In scope

- Луна: жёлтая, кратеры, не плоский белый шар.
- День: солнце с градиентом и ореолом; облака плотнее и мягче.
- Небо: **чередование** — большой дирижабль, малый дирижабль, широкофюзеляжный лайнер, частный самолёт, вертолёт (`prefers-reduced-motion` — статичный крупный дирижабль).
- Горизонт: промзона (доп. силуэты/трубы), окна **асимметрично** (часть тёмная, часть со светом); день — редкие «светящиеся» окна.
- Лейаут: дорога ниже, карточка выше, переключатель день/ночь выше; карточка стеклянная (не ломать без причины).
- Грузовик: полировка «рекламного» уровня, без надписей.

## Out of scope

- Воркер, PDF, КР Минтранс — после принятия визуала.

## Критерии готовности

- [x] Небо не перегружено (слоты флота ~30 с, `prefers-reduced-motion`).
- [x] Окна асимметричны (ночь/день).
- [x] `npm run build` без ошибок.
