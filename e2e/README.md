# E2E тесты (Playwright)

Проверка логина по ролям, создания ЭПЛ водителем и скриншотов (наше приложение + ЛК Такском).

**Важно:** админ-панель по ЭПЛ, паркам, сотрудникам и ТС — это **Такском** (epl.taxcom.ru). У нас свой сайт со своими ролями (админ, менеджер, водитель). См. корневой файл `НАШ_САЙТ_И_ТАКСКОМ_СХЕМА.md`.

## Установка

```bash
cd e2e
npm install
npx playwright install chromium
```

## Учётные данные

Скопируйте `.env.example` в `.env` и заполните:

- **Наше приложение:** `E2E_BASE_URL` — адрес нашего фронта. `E2E_ADMIN_*`, `E2E_DRIVER_*`, `E2E_MANAGER_*` — логин/пароль для входа **в наш сайт** (наша админка, портал водителя, панель менеджера).
- **Такском (epl.taxcom.ru):** `TAKSKOM_MECHANIC_*`, `TAKSKOM_MEDIC_*`, `TAKSKOM_DISPATCHER_*`, `TAKSKOM_ADMIN_*` — учётные данные для входа **в ЛК Такском** (там своя админ-панель: парки, сотрудники, ТС, ПЛ, подписание титулов). Скриншоты делаются после логина в epl.taxcom.ru, в т.ч. в профиле (где видна роль).

Файл `.env` не коммитить (добавлен в .gitignore).

## Запуск

1. Запустите backend (`http://localhost:5000`) и frontend (`http://localhost:3000`).
2. Из папки `e2e`:

```bash
# Все тесты
npm test

# Только наше приложение (логин админ/водитель/менеджер, скрины)
npm run test:taxi

# Водитель: создание ЭПЛ и скрин карточки
npm run test:driver

# Такском: логин механик/медик/диспетчер и скрины
npx playwright test taxcom-login-screenshots
```

Скриншоты сохраняются в `e2e/test-results/`. Отчёт: `npm run report`.

## Что проверяют тесты

- **taxi-app-flow:** логин в **нашем** приложении (admin / driver / manager), переход в панель, скриншот; у водителя — список ЭПЛ, открытие первого ЭПЛ, скрин карточки.
- **driver-epl-flow:** логин водителя **на нашем сайте**, создание ЭПЛ (пробег/топливо), переход в карточку ЭПЛ, скриншот.
- **taxcom-login-screenshots:** логин в **epl.taxcom.ru** под механиком, медиком, диспетчером (TAKSKOM_*), переход в профиль (где видна роль), скриншот.

Подписание титулов в нашем приложении делается через signer-client (КриптоПро), а не через браузер; в ЛК Такском можно вручную проверить подписание после того, как скрины покажут интерфейс под каждой ролью.
