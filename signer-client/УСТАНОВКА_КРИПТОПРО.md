# Установка КриптоПро и настройка автоматического подписания ЭПЛ

## Шаг 1: Установка КриптоПро CSP

1. **Скачай и установи КриптоПро CSP:**
   - Перейди на https://cryptopro.ru/products/cades/plugin-download
   - Скачай единый дистрибутив **КриптоПро CSP** (включает КриптоПро ЭЦП Browser plug-in)
   - Установи на ПК (требуются права администратора)

2. **Проверь установку:**
   - Открой **Панель управления** → **КриптоПро CSP**
   - Убедись, что криптопровайдер работает

## Шаг 2: Установка сертификатов (.cer файлы)

1. **Сертификаты должны быть в корне проекта** (рядом с `signer-client/`):
   - `Амиргамзаев.cer` (диспетчер)
   - `Поливода.cer` (медик или механик)
   - `Иванов.cer` (одна из ролей)
   - `Таов Инал Нарзанович.cer` (одна из ролей)

2. **Запусти скрипт установки:**
   ```cmd
   cd signer-client
   node install-certificates.js
   ```

   Скрипт автоматически:
   - Найдёт все `.cer` файлы в корне проекта
   - Установит их в хранилище Windows "Личное" (My)
   - Покажет результат установки

3. **Проверь установку сертификатов:**
   - Открой **Панель управления** → **КриптоПро CSP** → **Сертификаты**
   - Убедись, что все сертификаты отображаются в списке

## Шаг 3: Установка расширения КриптоПро в браузер

### Для Chromium-Gost (рекомендуется):

1. **Скачай Chromium-Gost:**
   - https://github.com/deemru/chromium-gost/releases/latest
   - Распакуй в папку (например, `C:\Program Files\Chromium Gost\`)

2. **Расширение КриптоПро ЭЦП Browser plug-in:**
   - При установке **КриптоПро CSP** (единый дистрибутив) расширение устанавливается автоматически
   - Если расширение не появилось, переустанови КриптоПро CSP с опцией "Browser plug-in"

3. **Проверь работу расширения:**
   - Открой Chromium-Gost
   - Перейди на страницу с тестовым подписанием (например, в Такском)
   - При нажатии "Подписать" должно появиться окно выбора сертификата

### Для обычного Chrome/Edge:

1. **Установи расширение вручную:**
   - Открой Chrome/Edge
   - Перейди в **Расширения** (chrome://extensions или edge://extensions)
   - Включи **Режим разработчика**
   - Если расширение не появилось автоматически, установи его из установочного пакета КриптоПро CSP

## Шаг 4: Настройка .env

В файле `signer-client/.env` укажи:

```env
# Путь к браузеру Chromium-Gost (если используешь)
CHROMIUM_GOST_PATH=C:\Program Files\Chromium Gost\chrome.exe

# Путь к вспомогательному exe для автовыбора сертификата (опционально)
# Если не указан, при подписании нужно будет вручную выбирать сертификат
CERT_PICKER_EXE=

# Режим отладки (0 = скрытый, 1 = видимый браузер)
TAXCOM_HEADLESS=0
```

## Шаг 5: Сопоставление сертификатов с ролями

Программа автоматически выбирает сертификат по роли:
- **Диспетчер** → сертификат для диспетчера (например, `Амиргамзаев.cer`)
- **Медик** → сертификат для медика (например, `Поливода.cer`)
- **Механик** → сертификат для механика (например, `Иванов.cer`)

**Важно:** При первом запуске подписания в диалоге КриптоПро нужно будет **вручную выбрать нужный сертификат** для каждой роли. После этого браузер может запомнить выбор.

## Шаг 6: Проверка работы

1. **Запусти программу:**
   ```cmd
   cd signer-client
   node sign.js
   ```

2. **Создай тестовый ЭПЛ** через сайт (как водитель)

3. **Наблюдай за процессом:**
   - Программа должна автоматически заполнить Т1, Т2, Т3
   - При подписании появится диалог выбора сертификата
   - Выбери нужный сертификат для каждой роли
   - После успешного подписания ЭПЛ будет готов

## Устранение проблем

### Расширение КриптоПро не работает:
- Переустанови КриптоПро CSP с опцией "Browser plug-in"
- Перезапусти браузер
- Проверь, что расширение включено в настройках браузера

### Сертификаты не видны в диалоге:
- Убедись, что сертификаты установлены в хранилище "Личное" (My)
- Проверь, что у сертификатов есть закрытый ключ (если нет - установи .pfx вместо .cer)
- Перезапусти браузер после установки сертификатов

### Диалог выбора сертификата не появляется:
- Проверь, что расширение КриптоПро установлено и включено
- Убедись, что используется Chromium-Gost или браузер с поддержкой КриптоПро
- Проверь логи программы на наличие ошибок
