# Быстрый старт: установка КриптоПро и сертификатов

## 1. Установка КриптоПро CSP (5 минут)

1. **Скачай единый дистрибутив КриптоПро CSP:**
   - https://cryptopro.ru/products/cades/plugin-download
   - Выбери версию для Windows
   - Установи (требуются права администратора)

2. **Проверь установку:**
   - Открой **Панель управления** → **КриптоПро CSP**
   - Если видишь настройки — всё ок

## 2. Установка сертификатов (2 минуты)

1. **Сертификаты уже в корне проекта:**
   - `Амиргамзаев.cer` (диспетчер)
   - `Поливода.cer` (медик)
   - `Иванов.cer` (механик)
   - `Таов Инал Нарзанович.cer` (дополнительный)

2. **Запусти установку:**
   ```cmd
   cd signer-client
   install-certs.cmd
   ```
   
   Или вручную:
   ```cmd
   node install-certificates.js
   ```

3. **Проверь установку:**
   - Открой **Панель управления** → **КриптоПро CSP** → **Сертификаты**
   - Должны быть видны все установленные сертификаты

## 3. Установка расширения КриптоПро в браузер

### Вариант А: Chromium-Gost (рекомендуется)

1. **Скачай Chromium-Gost:**
   - https://github.com/deemru/chromium-gost/releases/latest
   - Распакуй в папку (например, `C:\Program Files\Chromium Gost\`)

2. **Расширение устанавливается автоматически** при установке КриптоПро CSP

3. **Проверь работу:**
   - Открой Chromium-Gost
   - Перейди на любой сайт с подписанием
   - При нажатии "Подписать" должно появиться окно выбора сертификата

### Вариант Б: Обычный Chrome/Edge

1. **Расширение должно установиться автоматически** при установке КриптоПро CSP

2. **Если не появилось:**
   - Открой Chrome/Edge → **Расширения** (chrome://extensions)
   - Включи **Режим разработчика**
   - Проверь, что расширение КриптоПро ЭЦП Browser plug-in включено

## 4. Настройка .env

Открой `signer-client/.env` и проверь:

```env
# Путь к браузеру (если используешь Chromium-Gost)
CHROMIUM_GOST_PATH=C:\Program Files\Chromium Gost\chrome.exe

# Сопоставление сертификатов с ролями (опционально, для подсказок)
CERT_DISPATCHER=Амиргамзаев.cer
CERT_MEDIC=Поливода.cer
CERT_MECHANIC=Иванов.cer
```

## 5. Проверка работы

1. **Запусти программу:**
   ```cmd
   cd signer-client
   node sign.js
   ```

2. **Создай тестовый ЭПЛ** через сайт (как водитель)

3. **Наблюдай за процессом:**
   - Программа автоматически заполнит Т1, Т2, Т3
   - При подписании появится диалог выбора сертификата
   - **Выбери нужный сертификат для каждой роли:**
     - **Т1 (диспетчер)** → сертификат диспетчера (Амиргамзаев)
     - **Т2 (медик)** → сертификат медика (Поливода)
     - **Т3 (механик)** → сертификат механика (Иванов)

## Что делать, если не работает?

### Расширение КриптоПро не появляется:
- Переустанови КриптоПро CSP с опцией "Browser plug-in"
- Перезапусти браузер
- Проверь, что расширение включено в настройках браузера

### Сертификаты не видны в диалоге:
- Убедись, что сертификаты установлены (запусти `install-certs.cmd` снова)
- Проверь в **Панель управления** → **КриптоПро CSP** → **Сертификаты**
- Перезапусти браузер после установки

### Диалог выбора сертификата не появляется:
- Проверь, что расширение КриптоПро установлено и включено
- Убедись, что используешь Chromium-Gost или браузер с поддержкой КриптоПро
- Проверь логи программы на наличие ошибок
