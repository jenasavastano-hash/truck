# Программа на ПК клиники (одна точка входа)

Один скрипт на ПК с ключами:
1. **Заявки на создание ЭПЛ** — опрос заявок (режим «Наш API»), отчёт о создании (пока заглушка; позже — Playwright в ЛК Такском).
2. **Подпись титулов** — опрос титулов на подпись, подпись КриптоПро, отправка на бэкенд.

**Запуск:** `run-signer.cmd` или `node sign.js`. Рядом — `.env` (скопировать из `.env.example`).

**.env:** `API_URL`, `SIGNER_API_KEY` (тот же, что в backend/.env), `POLL_SECONDS=20`, при реальной подписи — `SIGNER_CMD` (путь к .exe КриптоПро).

**Один проход:** `node sign.js --once`.

**Exe:** `npm run build` — собирает один .exe для ПК без Node.
