## 1. Глоссарий
- **Сервис ЭПЛ** — продукт для создания электронных путевых листов и взаимодействия с ГИС ЭПД.
- **ЛК ЭПЛ** — личный кабинет для работы с путевыми листами.
- **ПЛ** — Электронный путевой лист (6 титулов, XML по формату ФНС).
- **Титул** — часть ПЛ (t1..t6).
- **ТС** — транспортное средство.
- **ВУ** — водительское удостоверение.

---

## 2. Описание сервиса
Кратко: система позволяет создавать и отправлять ПЛ в ГИС ЭПД, получать ответы и QR-код. Роли: Админ, Диспетчер, Медработник, Механик, Водитель. Титулы и их назначение описаны в общем документе.

## 3. Добавление страницы epl.taxcom.ru в избранное
1. Откройте браузер (например Chrome).
2. Перейдите на `https://epl.taxcom.ru`.
3. Нажмите значок «☆» в адресной строке и сохраните закладку.

---

## 4. Авторизация в системе
1. Перейти на `https://epl.taxcom.ru`.
2. Ввести логин и пароль, нажать «Войти».
3. При первой авторизации — смена пароля: заполнить текущий и новый пароль.
4. После входа откроется `Список путевых листов`.

---

## 5. Работа с фильтрами
- Нажать кнопку настройки фильтра → выбрать критерии через чекбоксы (Поиск по, Статус, Смена, Автопарк и т.д.).
- При необходимости задать диапазон дат.
- Для применения — нажать `Применить`, затем в поле `Поиск` ввести термин и нажать `Enter`.
- Очистка: Backspace или нажатие крестика рядом с критерием.
- Восстановление: `Вернуть поля по умолчанию` или `Добавить поле`.

---

## 6. Просмотр списка и детальной страницы ПЛ
- Откройте раздел `Электронные путевые листы`.
- Вкладки: `На подписании`, `Закрытые`, `Недействительные`, `Все ПЛ`, `Удаленные`.
- Нажмите на кликабельный номер ПЛ — откроется детальная страница с блоками состояния каждого титула и краткой информацией.
- Для подробностей титула нажмите `Открыть` — можно скачать XML и `.sig` (кнопка `Скачать`).

---

## 7. Создание нового путевого листа (Администратор)
> Примечание: админ может создать ПЛ, но не подписывать титулы.

1. В `Список путевых листов` нажмите `Создать`.
2. Откроется форма Титул 1 — заполните:
   - Выберите автопарк (поле `Автопарк`).
   - Номер путевого листа (waybillNumber) и дата (waybillDate).
   - Признак обязательности Титула 6 (если не нужен — выбрать `нет`).
   - Признак формирования путевого листа на один день: если `нет`, заполните `useWaybillDateFrom` и `useWaybillDateTo`.
3. Выберите ТС: введите гос.номер или инвентарный номер и выберите из списка. При отсутствии ТС заполните поля вручную.
4. Добавление прицепа: `Добавить` в блоке `Прицеп` — выбрать зарегистрированный или заполнить вручную.
5. Выберите/добавьте водителя: введите ФИО или табельный номер, выберите из списка или заполните вручную.
6. Блок `Подписант` оставьте пустым (подписание выполняют другие роли).
7. Нажмите `Сохранить`. ПЛ появится в `Все ЭПЛ` и у диспетчера в `На подписании`.

---

## 8. Удаление и восстановление ПЛ
- Удалить до статуса `QR-код получен`.
- Выберите ПЛ чекбоксом → нажмите `Удалить` → статус `Удален` (вкладки `Все ЭПЛ` и `Удаленные`).
- Для окончательного удаления: перейдите в `Удаленные`, выберите и нажмите `Удалить` — данные удаляются без восстановления.
- Восстановление: в `Удаленные` отметьте ПЛ и нажмите `Восстановить` — статус вернется к предыдущему.

## 9. Регистрация / редактирование автопарка
### 9.1 Регистрация автопарка
1. Перейдите в раздел `Перевозчик`.
2. Нажмите `Зарегистрировать автопарк`.
3. В модальном окне заполните:
   - Наименование автопарка
   - Юридический адрес
   - Тип транспортного средства (при необходимости)
4. Нажмите `Сохранить автопарк` — автопарк появится в списке.

### 9.2 Редактирование автопарка
1. Нажмите `три полоски` рядом с автопарком → `Редактировать`.
2. Внесите изменения → `Сохранить автопарк`.

### 9.3 Блокировка / активация автопарка
- Для отключения: `три полоски` → `Отключить` → статус `Неактивен`. Сотрудники автопарка не смогут авторизоваться.
- Для активации: `три полоски` → `Активировать` → статус `Активен`.

---

## 10. Регистрация / редактирование сотрудников автопарка
### 10.1 Регистрация сотрудника
1. Откройте детальную страницу автопарка → меню `Сотрудники`.
2. Перейдите на вкладку роли (Водитель / Механик / Медработник / Диспетчер и т.п.).
3. Нажмите `Зарегистрировать сотрудника`.
4. Заполните все поля в модальном окне и нажмите `Сохранить сотрудника`.

**Особенности:**
- Для `Медицинский работник` — обязательно указать данные лицензии (серия, номер, даты).
- Для `Водитель` — обязательно данные ВУ; если `иностранное ВУ` — проставить чекбокс, что позволит ввод буквенных значений в серию/номер.

### 10.2 Редактирование сотрудника
- `Три полоски` рядом с ФИО → `Обновить` → в модальном окне внести изменения → `Обновить сотрудника`.

### 10.3 Блокировка / активация сотрудника
- `Три полоски` → `Уволить` → статус `Уволен` (авторизация заблокирована).
- Для восстановления: `Три полоски` → `Активировать` → статус `Работает`.

### 10.4 Удаление сотрудника
- `Три полоски` → `Удалить` (следует подтвердить действие в модальном окне).

---

## 11. Регистрация / редактирование транспортного средства (ТС)
### 11.1 Регистрация ТС
1. На детальной странице автопарка перейдите в `Транспорт` → тип ТС.
2. Если тип ТС отсутствует — добавьте его через `+` в списке типов.
3. Нажмите `Зарегистрировать` внизу таблицы типа ТС.
4. Заполните обязательные поля:
   - `Регистрационный номер` (REG) и/или `Инвентарный номер` (inventoryNumber)
   - `Марка`
   - `Модель`
5. По желанию заполните блок `Сведения о топливе` (тип топлива, единица измерения, сезонность и т.д.). Эти данные подставляются в Титулы 4 и 5 при выборе данного ТС.
6. Нажмите `Сохранить` — ТС появится в списке.

### 11.2 Редактирование ТС
- `Три полоски` рядом с ТС → `Редактировать` → внести изменения → `Сохранить`.

### 11.3 Удаление ТС
- `Три полоски` → `Удалить` → подтверждение удаления.

---

## 12. Формирование отчетов
Все отчеты доступны в разделе `Отчет`.

### 12.1 Отчет по ЭПЛ
1. В `Отчет` → `Отчет по ЭПЛ`.
2. Заполните критерии: период (с/по), статусы ЭПЛ (множественный выбор), автопарк.
3. Нажмите `Применить` — результат в таблице.
4. Для скачивания — `Скачать` (файл Excel).

### 12.2 Отчет по рейсам
1. В `Отчет` → `Отчет по рейсам`.
2. Критерии: период (по умолчанию вчера), статусы, автопарк.
3. `Применить` → `Скачать` (Excel).

### 12.3 Журнал механика
1. В `Отчет` → `Журнал механика`.
2. Критерии: период (Диапазон/Дата), автопарк (по умолчанию автопарк пользователя), ТС (опционально; множественный выбор).
3. `Применить` → таблица → `Скачать` (Excel).
4. Для скачивания архива с `.sig` — `Скачать все файлы` → архив с папками по ТС (в каждой: Excel + .sig).

## Полезные примечания для интеграции
- При регистрации автопарка админ вводит полные данные: эти данные затем используются для синхронизации с API Такскома (создание парка через `createPark`).
- При регистрации сотрудников и ТС обязательно сохранять поля, требуемые для Такскома: ИНН, табельный номер, серия/номер ВУ, inventoryNumber/registrationNumber.
