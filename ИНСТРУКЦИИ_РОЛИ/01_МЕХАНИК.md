# 🔧 Инструкция для роли МЕХАНИК (Такском-Транспорт)

## 1. Заполнение и подписание Титула 3 – Технический контроль ТС

### 1.1 Открытие путевого листа
- В разделе «Список путевых листов» выбрать ПЛ
- Нажать на кликабельный номер
- Откроется детальная страница ПЛ

### 1.2 Заполнение Титула 3
- Нажать на кнопку «Заполнить»
- **Проверить автоматически заполненные поля**
- Заполнить пустые поля

### 1.3 Автоматическое заполнение полей
Поля подставляются автоматом:
- **«Дата проведения контроля»** - из файла, если был направлен; иначе текущая дата
- **«Время проведения контроля»** - из файла, если был направлен; иначе текущее время
- **«Дата и время выпуска ТС на линию»** - из файла, если был направлен; иначе текущие значения

### 1.4 Работа с выпадающими списками
- Нажать на поле
- Выбрать необходимый вариант

### 1.5 Результат контроля ТС
**Поле: «Отметка по результату проведения предрейсового/предсменного контроля ТС»**

**Если выбрать «Выпуск на линию не разрешен»:**
- Статус Титула 3 становится **«Недействителен»**
- Отправка в ГИС **не происходит**
- ПЛ переходит на вкладку **«Недействительные»**
- Статус ПЛ: **«Титул 3: Недействителен»**

**Если выбрать «Выпуск на линию разрешен»:**
- Можно подписать и отправить

### 1.6 Сохранение Титула 3
- Нажать кнопку **«Сохранить»** (если Титул 3 ещё не подписан)

### 1.7 Подписание Титула 3
- Нажать кнопку **«Подписать»**
- Введённые данные **автоматически сохраняются**
- **⚠️ После подписания изменения вносить НЕЛЬЗЯ**

### 1.8 Процесс подписания
- После нажатия **«Подписать»** → появляется окно **«Подтверждение доступа»**
- Нажать **«Да»**
- Произойдёт перенаправление на детальную страницу ПЛ
- В блоке с информацией статус изменится на **«Подписан»**
- Откроется доступ к кнопке **«Открыть»**

### 2.6 Подписание Титула 4
- Нажать **«Подписать»**
- Введённые данные **автоматически сохраняются**
- **⚠️ После подписания изменения вносить НЕЛЬЗЯ**

### 2.7 Процесс подписания Титула 4
- После нажатия **«Подписать»** → появляется окно **«Подтверждение доступа»**
- Нажать **«Да»**

### 2.8 Автоматическое изменение статуса
**Когда подписаны Титулы 1-4:**
- Статусы **всех титулов 1-4** автоматически изменяются на **«Отправлен в ГИС»**

## 4. Работа с Журналом Механика

### 4.1 Формирование журнала
В разделе **«Отчеты»** → **«Журнал механика»**

### 4.2 Критерии формирования
Необходимо заполнить:

1. **Период:**
   - Выбрать **«Диапазон»** (по умолчанию: вчера-сегодня)
   - ИЛИ **«Дата»** (по умолчанию: сегодня)

2. **Автопарк:**
   - По умолчанию подставляется автопарк пользователя
   - Можно выбрать другой

3. **Транспортное средство:**
   - Опционально - можно выбрать конкретное ТС
   - Если не заполнить - журнал сформируется по ВСЕм ТС за период

### 4.3 Выбор ТС
- Вписать данные ТС в поле
- Выбрать из выпадающего списка
- **Возможен множественный выбор ТС**
- Для удаления ТС из списка - нажать **«крестик»** рядом

### 4.4 Применение фильтра
- После заполнения всех критериев нажать **«Применить»**
- Внизу отобразится таблица журнала механика

### 4.5 Скачивание журнала
**Вариант 1 - Excel файл:**
- Нажать на кнопку **«Скачать»**
- Дождаться завершения скачивания
- Открыть файл

**Вариант 2 - Архив с подписями (.sig):**
- Нажать на кнопку **«Скачать все файлы»**
- Дождаться завершения скачивания архива
- Структура архива:
  - Папки с наименованиями каждого ТС
  - В каждой папке:
    - Excel файл строк журнала
    - Файл в формате **.sig** (подпись)

## 5. Ручное создание записи журнала механика

### 5.1 Создание новой записи
- На странице **«Журнал механика»** нажать **«Создать запись»**
- Откроется модальное окно

### 5.2 Автоматически заполняемые поля
Некоторые поля заполняются автоматом:
- **Автопарк** - текущий автопарк пользователя
- **Дата проведения контроля** - сегодняшняя дата
- **Время проведения контроля** - текущее время

### 5.3 Заполнение вручную
1. **Выбрать ТС:**
   - Выбрать из выпадающего списка
   - Поля **автоматически заполнятся**

2. **Отметка о прохождении контроля:**
   - Выбрать необходимый вариант

3. **Показание одометра:**
   - Ввести вручную

4. **Табельный номер водителя:**
   - Если водитель уже в системе - выбрать из списка
   - Если нет - ввести вручную

### 5.4 Проверка и подписание
- **Внимательно проверить ВСЕ введённые данные**
- Нажать **«Подписать»**
- Запустится процесс подписания данных
- **⚠️ После подписания изменения вносить НЕЛЬЗЯ**

### 5.5 Процесс подписания записи
- После нажатия **«Подписать»** → окно **«Подтверждение доступа»**
- Нажать **«Да»**
- Новая запись отобразится в таблице журнала

## 📋 КЛЮЧЕВЫЕ МОМЕНТЫ

✅ **Обязательно заполнить:**
- Показания одометра (Титулы 4, 5)
- Результат контроля ТС (Титул 3)
- Все поля в журнале механика

⚠️ **ВАЖНО:**
- После подписания титула **ИЗМЕНЕНИЯ ВНОСИТЬ НЕЛЬЗЯ**
- Если результат контроля = «не разрешен» → ПЛ станет недействительным
- Титулы 1-4 при подписании автоматически отправляются в ГИС
- Титул 5 подписывается кнопкой **«Подписать И ОТПРАВИТЬ»**

🔐 **Процесс подписания:**
1. Нажать «Подписать»
2. Появится окно подтверждения
3. Нажать «Да»
4. Статус изменится на «Подписан»
5. Для ПЛ могут измениться статусы

📊 **Журнал механика:**
- Можно скачать Excel или архив с подписями (.sig)
- Архив содержит папки по ТС
- В каждой папке - Excel и файл .sig
