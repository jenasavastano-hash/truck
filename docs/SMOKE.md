# Smoke checklist (перед релизом)

Короткая ручная проверка после деплоя или перед демо.

## Auth

- [ ] Логин под тестовым пользователем, выход, повторный вход.
- [ ] `/home` открывается, карточки ролей ведут на нужные маршруты.

## Водитель

- [ ] Кабинет водителя загружается, профиль открывается без белого экрана.
- [ ] Смена / ЭПЛ: основные кнопки откликаются (без 500 в консоли на счастливом пути).

## Менеджер / директор

- [ ] Выбор парка, вкладки (автопарк, водители и т.д.) переключаются.
- [ ] Выпадающие меню в шапке не ломают вёрстку.

## Админ

- [ ] Список парков, создание/редактирование без падения.
- [ ] Разделы ЭПЛ / касса открываются (если включены).

## API

- [ ] `GET /api/health` (или ваш health endpoint) — 200.
