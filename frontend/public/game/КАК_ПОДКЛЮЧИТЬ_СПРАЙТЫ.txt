КУДА КИДАТЬ:
  Все картинки — в папку  frontend/public/game/
  (рядом с этим файлом).

КАК ИГРА ИХ БЕРЁТ:
  Игра запрашивает файлы:  http://localhost:3000/game/имя.png
  В Vite всё из public/ отдаётся с корня → файл public/game/taxi.png
  в игре идёт как /game/taxi.png.

───────────────────────────────────────────────────────────────
  КАК ЗАГРУЗИТЬ СПРАЙТ ДОРОГИ И ЗЕБРЫ
───────────────────────────────────────────────────────────────

ДОРОГА:
  1. Положи файл с именем  road.png  в папку  frontend/public/game/
  2. Спрайт дороги в игре растягивается по полосам движения: рисуется
     только в зоне между обочинами и тайлится по вертикали (едет вниз).
  3. Рекомендуемый вид: полоска текстуры асфальта с разметкой полос
     (например тонкие линии между полосами). Высота может быть любой —
     игра сама масштабирует и повторяет по вертикали.

ЗЕБРА:
  1. Положи файл с именем  zebra.png  в папку  frontend/public/game/
  2. Зебра — это горизонтальные белые полосы поперёк дороги (пешеходный
     переход вид сверху). В игре картинка растягивается по ширине дороги
     и повторяется по вертикали, полосы «уезжают» вниз вместе с движением.
  3. Рекомендуемый вид: горизонтальные белые полоски на тёмно-сером фоне,
     одна «волна» зебры; по высоте — на твой вкус (игра масштабирует).

После добавления файлов перезапусти или обнови страницу (F5) в мини-игре.

У декораций в игре уже есть анимации:
  — светофор: мигает красный / жёлтый / зелёный;
  — кусты и деревья: лёгкое покачивание и пульс масштаба;
  — знак: лёгкое покачивание;
  — перекрёсток: лёгкое мерцание яркости.

Бусты (в игре и в меню магазина):
  boost_magnet.svg / boost_magnet.png   — магнит (притягивает монеты)
  boost_nitro.svg / boost_nitro.png     — нитро (ускорение)
  boost_jump.svg / boost_jump.png       — прыжок
  boost_extra_life.svg / boost_extra_life.png — доп. жизнь
  Игра сначала ищет .svg, потом .png. В меню те же файлы.
  В проекте уже лежат SVG в одном стиле (плоские иконки, прозрачный фон).
  Варианты PNG со светлым фоном (удобно обрезать фон в редакторе) лежат в:
  C:\Users\Данил\.cursor\projects\c-Users-Desktop-1\assets\
  — скопируй boost_*.png в эту папку (frontend/public/game/) и при необходимости
  убери белый фон (Прозрачность в Фотошопе/Photopea/remove.bg), чтобы в игре не было белого квадрата.

Если какого-то файла нет — игра рисует запасной вариант (фигуры).

───────────────────────────────────────────────────────────────
  ЧТО ДЕЛАТЬ ДАЛЬШЕ
───────────────────────────────────────────────────────────────

  1. Скопируй PNG в  frontend/public/game/
  2. Имена — ТОЧНО как в списке выше.
  3. Запускай игру через  npm run dev  в папке frontend, открой
     http://localhost:3000  (не открывай index.html двойным кликом).
  4. Зайди в игру водителя → мини-игра → F5, чтобы подтянуть спрайты.

ПРОЗРАЧНЫЙ ФОН (для машин, такси, монеты, знаков, кустов, дерева):
  Игра рисует PNG как есть. Чтобы не было белого/серого фона — сделай
  фон прозрачным: photopea.com (Magic Wand по фону → Delete) или
  remove.bg, сохрани как PNG с прозрачностью.

Дорога (road.png) и зебра (zebra.png) — фон может быть непрозрачным,
их игра кладёт на полосу и они перекрывают только свою зону.
