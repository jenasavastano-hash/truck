# 🚀 Инструкция по деплою на Netlify

## ✅ Что делать сейчас

Ты видишь меню выбора. Сделай следующее:

1. **Выбери первый вариант:** `⇄ Link this directory to an existing project`
   - Нажми стрелку вверх/вниз для выбора
   - Нажми Enter

2. **Выбери проект `tvoyputevoy`** из списка
   - Если список большой, используй стрелки для поиска
   - Нажми Enter

3. **Готово!** Проект связан

4. **Запусти снова:** `deploy.cmd`

---

## 🔄 После связывания

После того как проект связан, каждый раз просто запускай:

```bash
deploy.cmd
```

И всё! Сайт автоматически обновится.

---

## 📝 Альтернатива: Ручная команда

Если хочешь сделать вручную:

```bash
# Связать с проектом (один раз)
netlify link

# Выбрать проект "tvoyputevoy"

# Деплой
netlify deploy --prod
```

---

## ❓ Если не видишь проект в списке

1. Проверь, что ты залогинен: `netlify status`
2. Если проекта нет, создай новый:
   - Выбери `+ Create & configure a new project`
   - Введи имя: `tvoyputevoy`
   - Выбери папку: `dist`
